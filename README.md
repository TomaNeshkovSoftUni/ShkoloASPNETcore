# Школо (ShkoloASPNETcore) – Училищна Уеб Система

Това е уеб базирана платформа, която (се опитва да) клонира Shkolo. 

---

Основни Функционалности

* **Управление на Потребители и Роли:** Диференциран достъп до ресурсите на системата въз основа на роли – *Администратори*, *Учители* и *Ученици*.
* **Ученици и Класове:** Възможност за регистрация на ученици и структуриране на учебните профили.
* **Учебни Предмети:** Динамично създаване и редактиране на учебни предмети.
* **Въвеждане на Оценки:** Въвеждане, редактиране и изтриване на оценки за всеки ученик по съответния предмет.
* **Автоматичен Одит (Issuer):** Системата автоматично записва името/имейла на профила (Учител или Администратор), който е въвел или коригирал съответната оценка, за максимална прозрачност.

---

Технологичен стек

* **Backend:** .NET / ASP.NET Core (MVC / Razor Pages)
* **Database ORM:** Entity Framework Core (EF Core)
* **База Данни:** Microsoft SQL Server
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
* **Сигурност:** ASP.NET Core Identity (Удостоверяване и авторизация)
